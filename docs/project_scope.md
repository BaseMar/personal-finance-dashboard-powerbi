# Project Scope

## Objective

The objective of this project is to build an interactive Power BI dashboard for analyzing personal finances, including income, expenses, budget usage, savings and spending recommendations.

The dashboard is designed to help users better understand their financial behavior and make data-driven budgeting decisions.

## Target User

The target user is an individual who wants to monitor personal finances, identify spending patterns and improve savings habits.

## Key Business Questions

The dashboard should answer the following questions:

1. What are the total monthly income and expenses?
2. What is the monthly net cash flow?
3. Which expense categories have the highest impact on the budget?
4. How do expenses change month over month?
5. Is the user staying within the planned monthly budget?
6. What is the monthly savings rate?
7. Which categories offer the highest savings potential?
8. Are there any unusual or unexpected spending patterns?

## Key Features

- Monthly income and expense tracking
- Net cash flow analysis
- Expense analysis by category and subcategory
- Budget vs actual comparison
- Savings rate monitoring
- Spending trend analysis
- Recommendations for cost optimization
- Transaction-level analysis

## Out of Scope

The following features are not included in the first version of the project:

- Real-time bank account integration
- Automatic connection to banking APIs
- Investment portfolio analysis
- Tax calculations
- Multi-user authentication

## Tools

- Power BI
- Power Query
- DAX
- Excel / CSV
- Python
- Git / GitHub

## Success Criteria

The project will be considered successful when:

- The data model is clean and well-structured.
- The dashboard answers the main business questions.
- Core financial metrics are calculated using DAX.
- The project includes documentation in the repository.
- The dashboard contains clear insights and recommendations.
- The repository is ready to be shared as a portfolio project.

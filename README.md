# Personal Finance Intelligence Dashboard

## Project Overview

Personal Finance Intelligence Dashboard is a Power BI project designed to analyze personal income, expenses, budget usage, savings rate and spending patterns.

The goal of this project is to provide a clear overview of financial behavior and support better budgeting decisions through interactive reports and data-driven recommendations.

## Business Problem

Many people track their expenses manually, but they often lack a clear view of spending patterns, monthly trends, budget deviations and savings potential.

This dashboard aims to answer questions such as:

- How much do I earn and spend each month?
- What is my monthly net cash flow?
- Which categories generate the highest expenses?
- Am I staying within my planned budget?
- What percentage of income am I able to save?
- Which spending areas should be optimized?

## Project Goals

- Build an interactive Power BI dashboard for personal finance analysis.
- Create a clean and structured data model.
- Use Power Query for data cleaning and transformation.
- Implement DAX measures for financial analysis.
- Analyze income, expenses, budget usage and savings.
- Provide data-driven recommendations for cost optimization.
